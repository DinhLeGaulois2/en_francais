Day 4 - Add and Commit Changes, Then Push Your Changes From Local to Remote [Home]

Goals:

After modifying files, we want to update the content at the remote repository (at "Github"). 3 Steps:

  1. Stage your changes (ADD to INDEX)

    • A specific file

        git add my_file

    • Stages all changes

        git add -A

    • Stages new files and modifications, without deletionss

        git add .

    • Stages modifications and deletions, without new files

        git add -u
  2. Commit your changes (COMMIT to HEAD)

      git commit -m "any_comment_here"

  3. Push your changes from LOCAL to REMOTE

      git push origin master
