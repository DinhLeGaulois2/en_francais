Installation

  1. Install NodeJS

  2. Creation of "package.json" (in a "console"):

         npm init


  3. Installation of the "SASS" compiler:

         npm install node-sass --save-dev


  4. Compile Commande:

              

    In a console, to compile (from SCSS to CSS):

         npm run compile:sass
