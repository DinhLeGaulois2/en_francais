Installation and Configuration

[Home]

  1. Create the "package.json"

         npm init -y

  2. To be able to use "React" we need to install "react" and "react-dom"

         npm i react react-dom

  3. To be able to use "webpack" and a dev server

         npm i --save-dev webpack webpack-dev-server webpack-cli

  4. Install "Babel"

         npm i @babel/core babel-loader @babel/preset-env @babel/preset-react --save-dev

  5. Create ".babelrc" ("src/.babelrc") with

    {
    "presets": ["@babel/preset-env", "@babel/preset-react"]
}

  6. Install "SASS"

         npm install node-sass css-loader sass-loader style-loader --save-dev

  7. Create (at the root of the project) the "webpack.config.js"

    const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
    entry: './src/index.js', // we need to create the 'src' folder and the file 'index.js'
    output: {
        path: path.join(__dirname, 'dist'),
        filename: 'index_bundle.js'
    },
    module: {
        rules: [
            { // for "Babel"
                test: /\.js$/,
                exclude: /node_modules/,
                use: {
                    loader: 'babel-loader'
                }
            },
            { // for "SASS"
                test: /\.(s*)css$/,
                use: ['style-loader', 'css-loader', 'sass-loader']
            }
        ]
    },
    plugin: [
        new HtmlWebpackPlugin({
            template: './src/index.html'  // to create the "index.html" file AUTOMATICALLY for us
        })
    ]
}

  8. Scripts for "SASS" compilation (in package.json)

    "scripts": {
    "start": "webpack-dev-server --mode development --open --hot",
    "build": "webpack --mode production"
},

  9. Result:

    1: "dist" is created when ("production" mode):

          npm run build


    2: "App.js", a React component
    3: "index.html", used in "webpack.config.js"
    needs it (see "plugins")...
    4: "index.js", the "entry" of "webpack.config.js"
    5: "package.json"
    6: "webpack.config.js"
